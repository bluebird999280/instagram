1. 시작 날짜 : 2024년 5월 15일
2. 목표 기간 : 1개월 (2024년 6월 14일)


## 개발 범위

1. 프로필 보기, 검색
2. 팔로우, 팔로우 목록 추천
3. 피드 (생성, 댓글, 좋아요)
4. 릴스 (생성, 댓글, 좋아요)

6. 인증 (로그인, 회원가입, 로그아웃) - JWT
